```groovy

	dependencies {
	        implementation 'com.github.zhuguohui.Android-Javassist:dx:v1.0.1'
	         implementation 'com.github.zhuguohui.Android-Javassist:javassist-android:v1.0.1'
	}
```

Javassist-Android

Extended Javassist for Android.

------
